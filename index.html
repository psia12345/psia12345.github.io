about

Racetyping is a single or two player typing game. The words are randomly generated in each round. The objective of the game is to type as fast and as accurately as one can until the time runs out. When the game is over, player recives a summary of typing speed (WPM).

Tech Stack: HTML5, CSS, JavaScript, Socket.io, Express, Node.js
